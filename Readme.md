# Buzz Lightyer

- Neste projeto especificaremos uma classe para determinar as funcionalidades 
do Buzz Lightyer

- Há a possibilidade de criação de vários Buzz Lightyer idênticos

## Algumas funcionalidades:

- Capacete retrátil
- 6 frases distintas
- Dispara laser
- Braço articulado para golpes
- Abre as asas
